Estrutura Básica do HTML

Voltar

Um documento HTML possui uma estrutura composta pelas tags < html >, < head > e < body > e a instrução < !Doctype >.


Falando Sobre Tags*

Voltar

Todas as marcações são realizadas pelas tags. Os navegadores interpretam essas instruções e a partir disso renderizam uma pagina. A forma mais comum de utilização das tags, são como delimitadores de estilo ou de conteúdo, ou seja ela agrupa os conteúdos de acordo com o tipo, forma e formatações e com isso outras linguagem como por exemplo CSS podem aplicar estilizações a este conteúdo.


Atributos Básicos das Tags

Voltar

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Alguns exemplos de atributos são: acesskey, class, draggable, hidden, id, lang, style, tabindex e title.


Textos

Voltar

Texto é aquilo que o usuário vai ler. No caso tipografia. As principais tags de texto são: h1 a h6, p, br, blockquote, strong, i, u, sub e sup entre outras.


Listas Ordenadas (ol) e Não Ordenadas (ul)

Voltar

Uma lista ordenada é uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag < ol >. Cada item da lista começa com a tag < li > que segue uma sequência de 1, 2, 3 por diante.
Uma lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos) e começa com uma tag < ul >.


Links

Voltar

A tag link ou "âncora" é usada para você acessar uma outra página por exemlo. É representada pela tag < a > e a referência href acomapanhada do site para onde você deseja que o usuário seja direcionado.Pode ser usada também os atributos target: "blank" e "self" para referenciar o site numa nova janela ou aba dentro do navegador e "title" para ajudar na navegabilidade.
